# ft_02: Linq-Embed-Mistral Fine-Tuning — Results Summary

## What We Did

Fine-tuned **Linq-Embed-Mistral (7B)** on the **moral-to-fable retrieval task** using contrastive learning (in-batch negatives).

- **Architecture**: LoRA adapters (r=64, α=128) — only ~0.76% of parameters trained
- **Loss**: MultipleNegativesRankingLoss — for each moral, the correct fable is the positive; all other fables in the batch are negatives
- **Evaluation**: 5-fold cross-validation with **GroupKFold** (ensures morals shared across multiple fables never appear in both train and test)
- **Corpus size**: 709 Aesop fables, each paired with its true moral

Two document formats were tested:
- **Raw**: fable text only
- **Fable+summary**: fable text + a one-line proverb-style summary generated by Gemini (from Exp 07)

---

## Results

| Setting | MRR |
|---|---|
| Baseline Linq (raw) | 0.2097 |
| **ft_02 fine-tuned (raw)** | **0.3177** (+0.108) |
| Baseline Linq (fable+summary) | 0.3601 |
| **ft_02 fine-tuned (fable+summary, cross-eval)** | **0.4160** (+0.056) |

> Cross-eval = model trained on raw fables, evaluated against fable+summary corpus. No retraining needed.

### Per-fold breakdown (fable+summary cross-eval)

| Fold | Baseline MRR | Fine-tuned MRR | Δ |
|------|-------------|---------------|---|
| 0 | 0.3376 | 0.4243 | +0.087 |
| 1 | 0.3987 | 0.4461 | +0.047 |
| 2 | 0.4056 | 0.4504 | +0.045 |
| 3 | 0.3491 | 0.3962 | +0.047 |
| 4 | 0.3096 | 0.3629 | +0.053 |
| **Mean** | **0.3601** | **0.4160** | **+0.056** |

---

## Key Takeaways

1. Fine-tuning on raw fables (+10.8 MRR points) shows the model learns the moral-to-narrative mapping, not just keyword overlap.
2. The fine-tuned model **generalizes to fable+summary documents** without retraining — it benefits from the richer document representation even though it never saw summaries during training.
3. Best single configuration so far: **ft_02 on fable+summary corpus, MRR = 0.4160**.

## Next Step: ft_03 — Hard Negative Fine-Tuning

The current model only sees easy negatives (random fables from the same batch). The next experiment adds **hard negatives**: adversarially designed distractor morals from the MORABLES MCQA benchmark, which include fables whose morals are semantically close but wrong. This forces deeper semantic understanding and is expected to push MRR above 0.42.
